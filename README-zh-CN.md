# ZMEM - 浙江多模态民族志图谱

**许可证:** MIT License

## 概述
ZMEM 是一个完整的多模态民族志技术框架，具有可实施的规范和详细文档。其伦理协议为浙江社区设计，正等待通过研究合作伙伴进行实地应用。

[English](README.md) | [Português](README-pt-BR.md)

## 主要创新
- 仪式手势检测，精度达 ±0.3°
- 吴语方言处理与社区验证
- 可逆同意协议（基于区块链）
- 经典文本语义共振分析
